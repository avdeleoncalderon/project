# Análisis de Vehículos Usados

Aplicación web interactiva para explorar datos de vehículos usados en venta.

## Características

- Visualización interactiva de datos
- Histogramas de distribución
- Gráficos de dispersión con línea de tendencia
- Catálogo completo del dataset
- Estadísticas descriptivas
- Filtros y controles interactivos

## Requisitos

- Python 3.8 o superior
- Las librerías listadas en `requirements.txt`
